# Video Assets Directory

Upload your MP4 video files here for the Motion Lab showcase section.

## File Naming Convention
- Use descriptive names: `brand-name-commercial-type.mp4`
- Example: `lays-cinematic-spot.mp4`

## Recommended Video Specs
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (1080p) or higher
- Frame Rate: 24-60fps
- File Size: Optimized for web (under 50MB per video)
